mrusty. mruby safe bindings for Rust

Requirements

  • mruby

mrusty requires mruby compiled with fPIC. To compile and install mruby 1.2.0:

  • make sure you have Bison & Ruby installed
  • download the source
  • unzip and cd to mruby-1.2.0/
  • add the following lines to build_config.rb as in the # C compiler settings example:
conf.cc do |cc|
    cc.flags << '-fPIC'
end
  • run ./minirake

A safe struct for the mruby API. The struct only contains creation and desctruction methods. Creating an MRuby returns a MRubyType (Rc<RefCell<MRuby>>) which implements MRubyImpl where the rest of the implemented API is found.
