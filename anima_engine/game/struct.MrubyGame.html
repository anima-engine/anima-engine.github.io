A struct used to run games from mruby directly.

Make sure you point to an mruby file with a Game Class defined which implements a method update(dt), where dt is a Float representing the time since the last frame.

Examples

let game = MrubyGame::new(Path::new("game.rb"));

GameLoop::new(game).run();
